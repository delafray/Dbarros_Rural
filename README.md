# Galeria de Fotos - Gestão de Projetos

Sistema simples e eficiente para gestão de fotos de projetos com organização por tags e categorias.

## Funcionalidades Atuais

- **Galeria de Fotos**: Visualização e busca de fotos de projetos.
- **Tags de Busca**: Organização de fotos por categorias e tags customizáveis.
- **Gestão de Usuários**: Controle de acesso para Admin, Usuário e Visitante.
- **Integração Supabase**: Armazenamento seguro de fotos e dados.

## Tecnologias Utilizadas

- React + TypeScript
- Tailwind CSS
- Lucide React (Ícones)
- Supabase (Backend/Storage)

## Como Iniciar

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Configure o arquivo `.env.local` com suas credenciais do Supabase
4. Execute o servidor de desenvolvimento: `npm run dev`
